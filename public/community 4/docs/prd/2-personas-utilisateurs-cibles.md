# 2. Personas & Utilisateurs Cibles

## 2.1 Persona Élève — Koffi, 16 ans

| Attribut | Description |
|---|---|
| Objectifs | Comprendre ses cours, obtenir de l'aide rapidement, partager ses connaissances |
| Frustrations | WhatsApp est chaotique, peur du jugement en posant des questions en classe |
| Comportement | Consomme Instagram quotidiennement, à l'aise avec les réseaux sociaux |
| Besoin Clé | Une interface familière, rapide, sans friction pour poser et répondre à des questions |

## 2.2 Persona Enseignant — Mme Dupont, 38 ans

| Attribut | Description |
|---|---|
| Objectifs | Diffuser des ressources pédagogiques, suivre les difficultés de ses élèves |
| Frustrations | Emails ignorés, WhatsApp mélange vie pro/perso, manque de visibilité |
| Comportement | Utilise des outils numériques basiques, préfère la simplicité d'usage |
| Besoin Clé | Poster des annonces facilement, répondre aux questions des élèves depuis un espace dédié |

## 2.3 Persona Administrateur — Direction de l'Établissement

| Attribut | Description |
|---|---|
| Objectifs | Superviser l'usage de la plateforme, modérer les contenus, analyser les statistiques |
| Frustrations | Absence d'outils de pilotage numérique dans l'établissement |
| Comportement | Familier avec les tableaux de bord Excel/Google Sheets |
| Besoin Clé | Dashboard clair avec métriques d'engagement, modération simple et rapide |
