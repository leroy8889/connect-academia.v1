# 14. Glossaire

| Terme | Définition |
|---|---|
| **Feed** | Fil d'actualité principal affichant les publications des utilisateurs suivis + recommandées |
| **Post** | Publication : texte et/ou image créée par un utilisateur |
| **Like** | Expression d'appréciation d'une publication (représentée par un cœur ❤️) |
| **Swipe** | Geste tactile de glissement vers la droite sur une publication pour ouvrir les commentaires |
| **Soft Delete** | Suppression logique d'un enregistrement (`is_deleted=1`) sans effacement physique de la BDD |
| **KPI** | Key Performance Indicator — indicateur de performance clé affiché dans le dashboard |
| **AJAX** | Technique JavaScript permettant des requêtes HTTP asynchrones sans rechargement de page |
| **Liquid Glass** | Style visuel inspiré d'Apple, caractérisé par la transparence, le flou et les surfaces lumineuses |
| **PWA** | Progressive Web App — application web installable sur l'écran d'accueil d'un mobile |
| **Rate Limiting** | Limitation du nombre de requêtes par utilisateur/IP pour prévenir les abus |
| **FAB** | Floating Action Button — bouton circulaire flottant pour l'action principale (créer un post) |
| **Polling** | Technique de requêtes périodiques au serveur pour simuler des mises à jour en temps réel |
| **bcrypt** | Algorithme de hachage de mots de passe sécurisé et résistant aux attaques par force brute |
| **MVC** | Model-View-Controller — architecture logicielle séparant données, affichage et logique |
| **PDO** | PHP Data Objects — couche d'abstraction BDD PHP utilisée pour les Prepared Statements |
| **CSRF** | Cross-Site Request Forgery — attaque exploitant la session d'un utilisateur authentifié |
| **XSS** | Cross-Site Scripting — injection de code JavaScript malveillant dans une page web |
| **RGPD** | Règlement Général sur la Protection des Données (règlement européen UE 2016/679) |
